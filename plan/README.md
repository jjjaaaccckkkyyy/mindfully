# Cloud-Based Multi-Agent Platform

A scalable, multi-tenant AI agent platform for SaaS.

## Overview

Cloud-based multi-agent system that orchestrates specialized AI agents to handle diverse tasks. Built with the following tech stack:

- **API**: Express + tRPC (TypeScript)
- **Agent Framework**: LangChain/LangGraph
- **Orchestration**: LangGraph
- **Vector DB**: Qdrant
- **Compute**: EKS + EKS Anywhere (AWS + on-prem hybrid)
- **Inter-Agent**: A2A Protocol + MCP
